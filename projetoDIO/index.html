Tudo sobre a WEB

Inicio da Internet

A internet teve seu início nas décadas de 1960 e 1970, com o desenvolvimento de ARPANET nos Estados Unidos.

Criadores como Vinton Cerf e Robert Kahn foram fundamentais na concepção do protocolo TCP/IP, a base da comunicação online.

A popularização começou na década de 1990, marcada pelo surgimento da World Wide Web, tornando-se acessível ao público em geral.

Primeiros sites

Os primeiros sites na web, na década de 1990, eram simples páginas HTML com texto e links. Tim Berners-Lee, criador da World Wide Web, desenvolveu o HTTP e HTML como as bases.

Hoje a evolução é evidente com tecnologias como CSS para estilização, JavaScript para interatividade e frameworks como React e Angular.

Bancos de dados dinâmicos, como MySQL e MongoDB, são essenciais. Além disso, a segurança é reforçada com HTTPS. A constante inovação define a criação e manutenção dos sites na era digital atual.

HTML5

O HTML5 é a quinta versão da linguagem de marcação HTML, amplamente utilizada para estruturar conteúdo na web.

Introduzido em 2014, o HTML5 traz novos elementos semânticos, como header e section, melhorando a compreensão do conteúdo pelos motores de busca. Além disso, suporta multimídia nativa, como áudio e vídeo, sem a necessidade de plugins.

Essa atualização fortalece a capacidade de criar páginas web mais dinâmicas e interativas.

Tags

As tags no HTML são elementos fundamentais que definem a estrutura e o significado do conteúdo em uma página web.

Elas são envolvidas por colchetes angulares (" < > ") e geralmente aparecem em pares de abertura e fechamento. A tag de abertura indica o início de um elemento, enquanto a tag de fechamento denota o término. Por exemplo,

inicia um parágrafo e < / p > encerra.< / p >

CSS

CSS (Cascading Style Sheets) é uma linguagem de estilo utilizada para controlar a apresentação e o layout de documentos HTML. Ele define como os elementos HTML são exibidos na página, permitindo o design e a formatação visual.

JavaScript

JavaScript é uma linguagem de programação de alto nível usada para tornar as páginas web interativas.

Executado no navegador do usuário, ele permite manipular dinamicamente o conteúdo, responder a eventos e criar experiências mais envolventes, adicionando funcionalidade às páginas web.
